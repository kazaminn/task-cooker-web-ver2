# Phase 5D: Soft Launch チェックリスト（人間用）

## ログイン・認証

- [ ] Google OAuthでログインできる
- [ ] ログアウト→再ログインが正常に動作する
- [ ] テストユーザー（Opus/Haiku/Sonnet）でemail/pwログインできる

## プロジェクト

- [ ] プロジェクト一覧が表示される
- [ ] プロジェクトを新規作成できる
- [ ] プロジェクト名・概要を編集できる
- [ ] プロジェクトを削除できる

## タスク

- [ ] タスク一覧（リストビュー）が表示される
- [ ] カンバンビューに切り替えられる
- [ ] タスクを新規作成できる（タイトル・説明・ステータス・優先度・担当者）
- [ ] タスクのステータスを変更できる（order→prep→cook→serve）
- [ ] カンバンでドラッグ&ドロップができる
- [ ] タスクの詳細画面が表示される
- [ ] タスクを編集できる（タイトル・説明・優先度等）
- [ ] タスクを削除できる
- [ ] フィルタ（ステータス・優先度・担当者・検索）が動作する
- [ ] displayId（#1, #2...）が正しく採番される

## コメント

- [ ] タスク詳細からコメントを投稿できる
- [ ] コメント一覧が表示される

## ダッシュボード

- [ ] ダッシュボードが表示される
- [ ] 注意セクション・最近のプロジェクト等が表示される
- [ ] クイック追加が動作する

## チーム

- [ ] チームメンバーが表示される
- [ ] チーム内のプロジェクトのみが見える（権限分離）

## UI・UX

- [ ] ダークモード切り替えが動作する
- [ ] モバイル幅でレイアウトが崩れない
- [ ] サイドバーの開閉が動作する
- [ ] ページ遷移が正常に動作する（戻る・進む含む）
- [ ] ローディング状態が表示される（白画面にならない）
- [ ] エラー時にメッセージが表示される

## パフォーマンス

- [ ] 初回ロードが極端に遅くない（体感5秒以内）
- [ ] タスク操作後のリアルタイム反映が動作する
- [ ] コンソールに致命的なエラーが出ていない
